# ImageComparision

This is a tool made to automate gui browser compatibility testing.

## Installation

### Requirements
* Python 3.3 and up
* OpenCV

## Usage

```python
mse(baseimg, qimg) # gives mse
histogram(baseimg, qimg) # gives histogram comparision
sim(baseimg, qimg # gives sim
```
